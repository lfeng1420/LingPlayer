## 简介

轻灵音乐是一款本地音乐播放器，支持 FLAC、APE、DSD、MP3、AAC 等常见音频格式，支持 CUE 分轨解析、分组随机播放。配合实时渲染的可视化频谱，以及透明效果与自定义背景的界面设计，带来丰富的个性化体验。

## 核心功能

- 支持 FLAC/APE/DSD/MP3/AAC 等数十种音频格式
- CUE 分轨文件智能解析
- 实时渲染的可视化频谱效果
- 透明毛玻璃效果 + 自定义背景图片
- 分组随机播放
- Steam 丰富状态显示
