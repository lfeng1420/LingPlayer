# 輕靈音樂

## 簡介

輕靈音樂是一款跨平台本地音樂播放器，支援 FLAC、APE、DSD、MP3、AAC 等數十種常見音訊格式，具備 CUE 分軌解析、分組隨機播放等實用功能。配合即時渲染的可視化頻譜與透明毛玻璃介面，搭配自訂背景圖片，帶來豐富的個人化體驗。

## ✨ 核心功能

- 支援 FLAC/APE/DSD/MP3/AAC 等數十種音訊格式
- CUE 分軌檔案智慧解析
- 即時渲染的可視化頻譜效果
- 透明毛玻璃效果 + 自訂背景圖片
- 分組隨機播放
- Steam 豐富狀態顯示
