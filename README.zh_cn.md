# 轻灵音乐

轻灵音乐是一款跨平台本地音乐播放器，支持 FLAC、APE、DSD、MP3、AAC 等数十种常见音频格式，具备 CUE 分轨解析、分组随机播放等实用功能。配合实时渲染的可视化频谱与透明毛玻璃界面，搭配自定义背景图片，带来丰富的个性化体验。

## ✨ 核心功能

- 支持 FLAC/APE/DSD/MP3/AAC 等数十种音频格式
- CUE 分轨文件智能解析
- 实时渲染的可视化频谱效果
- 透明毛玻璃效果 + 自定义背景图片
- 分组随机播放
- Steam 丰富状态显示
